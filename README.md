# practicethreelittlepigs
Tools: Git y GitHub
Deberás crear un repositorio de Github llamado "Practice-Three-Little-Pigs" en el cual deberás subir el ejercicio realizado en tu repositorio local.

Requisitos:
- Mediante código html deberás contar la historia de los tres cerditos.
- Deberás realizar los commits pertinentes y pushearlos al repositorio (OJO - El cuándo y el porqué de los commits es importante)
- Puedes darle alegría a tu historia con imágenes
- Por supuesto puedes darle estilos con CSS
- Activar Github Pages

Entregable:
- Enlace al repositorio de Github
- Una captura de pantalla del resultado
- Enlace al Github Pages.
